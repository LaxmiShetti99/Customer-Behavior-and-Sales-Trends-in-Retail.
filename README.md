# Customer Behaviour and Sales Trends in Retail
# Overview 
This project explores retail customer data to identify purchasing patterns, satisfaction levels, and revenue drivers. It combines SQL queries for structured data analysis, Python (pandas, matplotlib, seaborn) for statistical exploration and visualizations, and Power BI dashboards for interactive reporting.  
Dataset Source: Downloaded from Kaggle.

# Features
- SQL queries for data extraction and filtering  
- Python scripts for cleaning, preprocessing, and visualization  
- Correlation heatmaps to show variable relationships  
- Scatterplots for review ratings vs. spending  
- Heatmaps of purchases by location and category  
- Power BI dashboards for interactive insights  
- Business interpretations and recommendations  

# How to Run 
1. Clone the repository  
2. Install dependencies (`pandas`, `matplotlib`, `seaborn`)  
3. Run SQL scripts to query the dataset  
4. Use Jupyter notebooks for analysis and visualizations  
5. Open Power BI files to explore interactive dashboards

# Key Insights
- Weak correlation between ratings and spending  
- Regional differences in category demand  
- A small group of top customers drive most revenue  
- Discounts and promo codes influence purchases differently

# Tech Stack
- Python (pandas, matplotlib, seaborn)
- SQL (Structured Query Language)
- Power BI  
- Kaggle dataset

# Files
- project-1.py → Python code for data analysis
